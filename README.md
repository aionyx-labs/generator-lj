# OMR Answer Sheet Generator

An easy-to-use, browser-based web application designed to generate and print customized Optical Mark Recognition (OMR) / exam answer sheets. Perfect for teachers and schools to create quick answer sheets for Multiple Choice, Fill-in-the-blank (Short Answer), and Essay exams.

## Features
* **Customizable School Header:** Dynamically change school names and upload your own school logo.
* **Flexible Paper Sizes:** Supports both standard A4 and F4 (Folio) paper sizes.
* **Multiple Layout Sections:** Toggle and configure Multiple Choice questions (up to 20-40 questions with A-C or A-D options), Short Answer sections, and Essay sections with clean writing lines.
* **Print Ready:** Optimized with CSS print styles to ensure clean, perfect alignment when printing or saving as PDF.
* **100% Client-Side:** Runs completely in the browser without any database or external server tracking.

# Generator Lembar Jawaban Ujian

Aplikasi berbasis web yang ringan dan mudah digunakan untuk membuat dan mencetak Lembar Jawaban Komputer (LJK) atau lembar jawaban ujian yang dapat disesuaikan. Sangat cocok bagi guru dan sekolah untuk menyediakan lembar jawaban Pilihan Ganda, Isian Singkat, maupun Essay secara instan.

## Fitur Utama
* **Kop Sekolah Dinamis:** Mengubah teks nama baris kop sekolah secara langsung dan mendukung unggah logo sekolah sendiri.
* **Pilihan Ukuran Kertas:** Mendukung format ukuran kertas standar A4 dan F4 (Folio).
* **Konfigurasi Soal Fleksibel:** Pengaturan jumlah soal Pilihan Ganda (hingga 20-40 nomor dengan opsi jawaban A-C atau A-D), bagian Isian, serta bagian Essay lengkap dengan garis bantu menulis.
* **Siap Cetak:** Dilengkapi setelan CSS khusus cetak (*print mode*) untuk memastikan tata letak tetap rapi dan presisi saat diprint atau disimpan ke PDF.
* **Murni Client-Side:** Berjalan sepenuhnya di browser pengguna tanpa memerlukan database atau penyimpanan server eksternal.
